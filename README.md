# prompt-engineering
Estudo sobre Engenharia de Prompts

### Dicas
Usar o parâmetro de Temperatura abaixo de 1. Pois acima desse valor irá fazer com o que o modelo retorne com respostas dinâmicas, imprecisas, incoerentes e fora de contexto. Entre 0 e 1 as respostas tendem a ser mais precisas e consistentes. Quando maior a temperatura mais dinâmica será a resposta, quanto menor a temperatura mais precisa a resposta.
